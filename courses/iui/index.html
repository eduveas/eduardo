706.022: Intelligent User Interfaces

This course is about interfaces that collect data from the user and create models to anticipate or adapt the interaction accordingly. The course offers an opportunity to dive into machine learning topics applied to user interfaces. In doing so, the course introduces important concepts inherent to intelligent systems that interact with humans: interactive machine learning, interfaces to machine learning and active learning systems. Students tackle the issue of building an interface using algorithms that beyond learning to solve a task, use machine intelligence to learn about the human and the human behaviour. Gradual introduction to the topics of human-aware interaction.

The student taking this course will learn through experience to:

  • understand how too collect data with a user interface
  • understand how to apply machine learning algorithms to interfaces
  • have an insight about how modern computing systems collect and use data from humans

    • More importantly, from experience, the student will learn to formulate data collection strategies to jumpstart an intelligent interface and to apply machine learning to continuously improve the interaction.

Eduardo Veas, Univ. Prof.:

Eduardo is Professor for Intelligent and Adaptive User Interfaces at the Institute of Interactive Systems and Data Science, Graz University of Technology (ISDS TUGraz) and he is also Research Manager of the Knowledge Visualisation group at Know-Center GmbH. His interests lie in the field of intelligent interactive systems, in particular studying human aware computing and cognitive aspects of data analytics, virtual and augmented reality interfaces. Eduardo obtained his PhD in Computer Science in 2012 from Graz University of Technology, his Msc in Information Science and Technology from Osaka University, and holds a degree in Software Engineering from National University of Technology, Argentina.

Grading

20% participation 20% critique 60% project topic

The course is project based. Projects should be carried out in groups of up-to 3 people. Projects will require the existence of an App with a user interface, or some method to collect data from human participants and deliver feedback. Students are encouraged to reuse applications or devices built in other classes: User Interfaces, Visual Analytics, Game Design, etc. A collection of alternative projects are available within the Intelligent User Interfaces Lab of Prof. Veas. For the course, the student will:

  • choose a project topic to build an intelligent interface
  • propose a data collection strategy
  • desing the algorithm based intelligence
  • build and showcase a demo of the system

What this course is NOT

This course is not an introduction to HCI nor to visualization. It will not offer the possibility to design a user interface, an application, nor a visualization. For general courses on the topics covered visit:

  • User Interfaces (Eduardo Veas)
  • Evaluation Methods (Eduardo Veas)
  • HCI (Keith Andrews)
  • Visual Analytics (Sabol, Schreck, Veas)
  • Augmented Reality (Dieter Schmalstieg)

This course will not deal with building a user interface or application from scratch. Students are advised to use prototypes built in previous courses. Neither will this course provide detailed information on experimental designs for data collection. Instead the course concentrates on providing a solid background on methods and techniques behind intelligent user interfaces including data collection strategy, algorithm, feedback (adaption or direct interaction with human). This course is no replacement for a machine learning course.

Who should attend this course

This course is particularly suited for students interested on topics related to human machine interaction and artificial intelligence. It is a compulsory course in the Intelligent Systems (major) and an elective course for Games Engineering, Interactive and Visual Information Systems and Robotics.
